Сложность/Качество кода

  • Logical Lines of Code
  • Statements per function (12 и меньше - ОК)
  • Cyclomatic complexity (5 и меньше - ОК)
  • Nested loops (2 и меньше - ОК)

Halstead complexity measures

  • Difficulty — на сколько сложно понимать, развивать и ревьюить такой код
            difficulty = (#distinct_operators / 2) *
                         (#operands / #distinct_operands)

Halstead complexity measures

  • Volume — количество кода
            volume = (#operators + #operands) *
                     log2(#distinct_operators + #distinct_operands)

Halstead complexity measures

  • Effort — усилие на создание такого кода
            effort = difficulty * volume

Halstead complexity measures

  • Time — время на написание всего кода, секунд
            time = effort / 18

Halstead complexity measures

  • Bugs — оценка количства скрытых багов в коде
            bugs = pow(effort, 2/3) / 3000
